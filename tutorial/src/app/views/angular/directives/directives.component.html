Привязки данных

Что это такое?

Привязки данных - выражения которые встраиваются в шаблоны и вычисляются для генерирования динамического контента в документах HTML

Для чего они нужны ?

Привязки данных создают связь между элементами HTML в документах HTML и файлами шаблонов с данными и кодом в приложении

Как они используются?

Привязки данных применяются как атрибуты элементов HTML или специальные последовательности символов в строках

Основные встроенные директивы

ngClass Директива используется для назначения классов управляющим элементам
ngStyle Директива используется для назначения отдельных стилей
ngIf Директива вставляет контент в документ HTML при условии, что результат вычисления заданного выражения равен true
ngFor Директива вставляет контент в документ HTML для каждого элемента в источнике данных
ngSwitch ngSwitchCase ngSwitchDefault Директивы используются для выбора блоков контента, вставляемых в документ HTML, в зависимости от значения выражения
ngTemplateOutlet Директива используется для повторения блока контента
